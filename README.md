## ⚙️ How It Works

1. User selects **Rock, Paper, or Scissors**
2. Computer randomly selects its move
3. Winner is determined based on classic rules:

   * Rock beats Scissors
   * Paper beats Rock
   * Scissors beats Paper
4. Score updates automatically
5. Reset button clears scores and results
